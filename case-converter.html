Case Conversion Guide

  • camelCase: Variables in JavaScript, Java (e.g., userName)
  • PascalCase: Class names, React components (e.g., UserProfile)
  • snake_case: Variables in Python, Ruby (e.g., user_name)
  • kebab-case: URLs, CSS classes (e.g., user-profile)
  • CONSTANT_CASE: Constants in most languages (e.g., MAX_SIZE)
