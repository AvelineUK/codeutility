CSS Grid Quick Reference

Grid Properties:

  • grid-template-columns: Defines column tracks
  • grid-template-rows: Defines row tracks
  • gap: Shorthand for row-gap and column-gap
  • fr unit: Flexible fraction of available space
  • minmax(): Min and max size for tracks
  • auto-fit: Creates responsive grids

Tip: CSS Grid is perfect for two-dimensional layouts!
