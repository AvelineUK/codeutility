About ISO 8601

ISO 8601 is the international standard for date and time representation. It's used for:

  • API responses and requests
  • Database timestamps
  • Log files and system records
  • Data interchange between systems
  • Avoiding timezone ambiguity

Format: YYYY-MM-DDTHH:mm:ss.sssZ where T separates date and time, and Z indicates UTC.
