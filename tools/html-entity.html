About HTML Entities

HTML entities are used to display reserved characters in HTML. Common uses include:

  • Displaying special characters like <, >, &, "
  • Preventing XSS vulnerabilities by encoding user input
  • Displaying symbols and accented characters
  • Ensuring proper rendering across different browsers

Examples: < becomes &lt;, > becomes &gt;, & becomes &amp;

All tools run entirely in your browser. No data is collected or sent to any server.
