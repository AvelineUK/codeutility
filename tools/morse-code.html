About Morse Code

Morse code is a method of encoding text characters using sequences of dots (.) and dashes (-). Key features:

  • Letters separated by spaces
  • Words separated by forward slashes (/)
  • Dot (.) = short signal, Dash (-) = long signal
  • Supports A-Z, 0-9, and common punctuation
  • Used in radio communication and emergency signaling
