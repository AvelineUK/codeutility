About ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard for electronic communication. Each character is represented by a number from 0-127:

  • 32-126: Printable characters (letters, numbers, symbols)
  • 0-31: Control characters (newline, tab, etc.)
  • 65-90: Uppercase letters (A-Z)
  • 97-122: Lowercase letters (a-z)
  • 48-57: Digits (0-9)

Example: "Hello" = 72 101 108 108 111 (decimal)
