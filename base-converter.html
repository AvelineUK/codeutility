About Number Bases

A number base (or radix) determines how many unique digits are used to represent numbers:

  • Binary (Base 2): Uses 0-1
  • Octal (Base 8): Uses 0-7
  • Decimal (Base 10): Uses 0-9 (standard)
  • Hexadecimal (Base 16): Uses 0-9, A-F
  • Base 36: Uses 0-9, A-Z (max supported)

Example: 255 (decimal) = FF (hex) = 377 (octal) = 11111111 (binary)
